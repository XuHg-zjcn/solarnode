# 太阳能节点
将光伏获得的电能通过总线传输，不需要电池
目前只支持升压，不支持降压，不支持关闭输出

为了防止通信线(RS485)误接电源线造成过压等原因导致芯片等元器件损坏，使用了电容隔离直流电，数据必须经过直流平衡的编码后再传输，会失去一些兼容性。

## 目标功能
- MPPT
- RS485通信
- 电流电压数据汇报
- 光照、温度传感器
- 一个按键和指示灯
- UART和I2C接口，可连接其他传感器或进行调试
